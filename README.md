# Prime Routes Logistics - Website Clone

A premium, responsive logistics website with smooth animations, interactive features, and modern UI/UX.

## 🚀 Features

- **Responsive Design** — Mobile-first, works on all devices
- **Scroll Animations** — Intersection Observer-powered reveal effects
- **Counter Animation** — Animated stats with easing
- **Contact Form** — Validated with localStorage persistence
- **Shipment Tracking** — Interactive tracking input with demo responses
- **3D Card Tilt** — Hover-based perspective effects
- **Preloader** — Smooth loading animation
- **Mobile Menu** — Slide-in navigation with overlay
- **Back to Top** — Smooth scroll button
- **Parallax Effects** — Floating shapes with scroll

## 🌐 Deploy to Vercel

### Option 1: Vercel CLI
```bash
npm i -g vercel
vercel
```

### Option 2: GitHub + Vercel
1. Push to GitHub
2. Import in [vercel.com/new](https://vercel.com/new)
3. Framework: **Other**
4. Deploy — done!

### Option 3: Drag & Drop
1. Go to [vercel.com](https://vercel.com)
2. Drag the project folder into the dashboard

No build step needed — it's pure HTML/CSS/JS.

## 🛠 Tech Stack

- HTML5, CSS3 (custom, no frameworks)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter, Outfit)
- Font Awesome 6 icons
- Intersection Observer API
